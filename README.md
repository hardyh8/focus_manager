# Focus Manager
---

### **Description**

Focus Manager is a **Focus session App** built in **Flutter, Bloc, Drift, and Clean Architecture**. It provides:

-   **Focus session**.
-   **Calendar Task Management**.
-   Integrated with **Deep Work Analytics** for improvements.
